desing / web development

when:

main concept

Simple photography portfolio, very clear, on white background with photo gallery on the first page. Gentle animations in good taste. Static website that serves as a business card.

development

The static website has been written in HTML5 and SASS. Animations has been realized with the use of GSAP library. The biggest problem was to create a pictures gallery with the ability to preview photo. In the mobile version, the events had to be programmed to execute the user's gestures, e.g. zoom or image swap. All events has been developed in JavaScript.
