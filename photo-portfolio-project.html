main concept

Simple photography portfolio, very clear, on white background with photo gallery on the first page. Gentle animations in good taste.

development

The website is a static webpage and was written in HTML5 with styling in SASS. Animations realized with the use of GSAP library. The biggest problem was to create a pictures gallery with the ability to preview photo. In the mobile version, the events had to be programmed to execute the user's gestures, e.g. zoom or image change. All was developed in JavaScript.
